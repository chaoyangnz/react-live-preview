# Changelog

## [1.0.0] - 2026-03-08

### Added
- Live preview of React/JSX components in a side panel
- Automatic CDN resolution for npm imports (React, Recharts, D3, Three.js, and more)
- Configurable reload mode: `onChange`, `onSave`, or `off`
- Warning panel for unknown packages with unpkg fallback
- `Ctrl+Shift+V` / `Cmd+Shift+V` keyboard shortcut
- Editor title bar button for quick access
